# buffer.lol

A static landing page for buffer.lol, a media diagnostics concept focused on stream buffering, video latency, CDN health, and upload processing.

## Files

- `index.html` - page markup and metadata
- `style.css` - visual design and responsive styles
- `app.js` - interactions, waitlist modal, animation, and preview signup behavior
- `assets/` - dashboard images and favicon

## Notes

The waitlist form currently stores submissions in browser `localStorage` unless a real signup endpoint is added to `data-signup-endpoint`.
