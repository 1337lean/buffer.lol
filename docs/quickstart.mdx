# Quickstart

Run the public tester from any modern browser.

## 1. Open the tester

Go to [buffer.lol](https://buffer.lol) and choose **Start test**.

## 2. Pick a test size

The default **Standard** profile is usually enough. Use **Light** on metered connections and **Heavy** when you want a larger sample.

## 3. Read the report

The report combines four measurements:

| Metric | What it tells you |
| --- | --- |
| Download | How quickly your browser receives data |
| Upload | How quickly your browser sends data |
| Ping | How responsive the connection is |
| Jitter | How much latency changes between samples |

## 4. Repeat where it matters

Run one test near your router and another where you actually work, stream, or play. If the numbers change sharply, the issue is probably local Wi-Fi quality rather than the internet plan itself.

<Tip>
  Keep VPNs, cloud backup tools, and large downloads in mind. They can change the result even when your provider connection is healthy.
</Tip>
